# Cyberbullying Detection using BERTweet and Swarm Intelligence

## Overview
This project presents a cyberbullying detection framework using **BERTweet embeddings combined with Swarm Intelligence optimization techniques**.  
The system analyzes social media text and classifies whether the content contains cyberbullying.

The goal of this research is to improve the detection accuracy of harmful online behavior using advanced NLP models.

---

## Features
- Cyberbullying detection from social media text
- BERTweet-based feature extraction
- Swarm intelligence optimization
- Interactive HTML dashboard for results
- Automated report generation
